# Vela App — Assets Folder

Place any additional assets here:

assets/
  fonts/        → Custom font files (woff2, ttf)
  icons/        → PNG/SVG icons
  animations/   → Lottie JSON files for onboarding, success states

The app currently loads Google Fonts (Sora + Inter) via CDN.
To use locally, download the woff2 files and update the @font-face rules in style.css.
